# 赛力斯知识门户对外接口文档

根据 Excel 文件《赛力斯知识门户对外接口v1.1_20250912.xlsx》的前 3 个 Sheet 整理。

## 1. 文件列表接口

### 接口概览

- 接口名称：文件列表接口
- 接口地址：`/api/knowledge/webservice/getBillList`
- 传参方式：`formdata`

### 请求参数

| 字段名 | 描述 | 是否必填 | 备注说明 |
| --- | --- | --- | --- |
| `communityId` | 密钥 | 是 | 密钥 |
| `fdCateId` | 所属分类 ID | 否 | 所属分类 ID，传入则只取此目录下的分类 |
| `type` | 板块名称 | 否 | 暂时只支持：`mutildoc` |
| `pageno` | 页码 | 是 | 当前页码 |
| `pagesize` | 每页数据条数 | 是 | 数据条数 |
| `beginTime` | 更新时间 | 否 | 取这个时间之后更新的数据，用于增量数据获取时传此参数 |

### 鉴权说明

若知识门户系统存在接口访问验证，需在 Header 中增加 `Authorization` 参数，Excel 中给出的示例代码如下：

```java
HttpHeaders headers = new HttpHeaders();
byte[] encodedAuth = org.springframework.security.crypto.codec.Base64.encode(
    "知识门户接口账号:知识门户接口密码".getBytes(Charset.forName("UTF-8"))
);
String authHeader = "Basic " + new String(encodedAuth);
headers.set("Authorization", authHeader);
```

### 参考报文

```text
type=mutildoc&communityId=1900b384c2b4e4bc61d36b648bda887a&pageno=1&pagesize=5&beginTime=2020-01-01
```

### 请求参数

| --- | --- | --- | --- |
| `communityId` | 密钥 | 是 | 密钥 |

| `fdNo` | 文件编号 | 否 | 文件编号，与文件 ID 至少传一个 |

若知识门户系统存在接口访问验证，需在 Header 中增加 `Authorization` 参数，Excel 中给出的示例代码如下：

```java
);
String authHeader = "Basic " + new String(encodedAuth);
headers.set("Authorization", authHeader);
```

## 3. 附件获取接口

- 接口名称：附件获取接口
- 接口地址：`/api/knowledge/webservice/getAttachment`
- 传参方式：`formdata`

### 请求参数

| 字段名 | 描述 | 是否必填 | 备注说明 |
| --- | --- | --- | --- |
| `communityId` | 密钥 | 是 | 密钥 |
| `fdId` | 附件 ID | 是 | 文件 ID |

### 鉴权说明

若知识门户系统存在接口访问验证，需在 Header 中增加 `Authorization` 参数，Excel 中给出的示例代码如下：

```java
HttpHeaders headers = new HttpHeaders();
byte[] encodedAuth = org.springframework.security.crypto.codec.Base64.encode(
    "知识门户接口账号:知识门户接口密码".getBytes(Charset.forName("UTF-8"))
);
String authHeader = "Basic " + new String(encodedAuth);
```
